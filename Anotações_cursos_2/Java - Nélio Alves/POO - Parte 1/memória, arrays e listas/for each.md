## Aula97 - "Laço for each"
#forEach

**É uma sintaxe simplificada para percorrer coleções**
sintaxe básica:
```java
for (tipo apelido: colecção){
	<comando1>
	<comando2>
}
```

#### _Exemplo:_
```java
String[] vect = new Srtring[] {"Maria", "Bob", "Alex"};

for (String obj : vect){
	System.out.println("obj")}
```
